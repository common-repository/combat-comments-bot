== Description ==

This plugin avoids bots posting comments directly on your site.

= How to use =
After installing and activating the plugin, go to Settings->General and enter your unique `Combat Comment Bot Secret Key`

= How it Works =
This plugin adds [nonces](http://markjaquith.wordpress.com/2006/06/02/wordpress-203-nonces/  "nonces") using the secret key to the comment form for unique validation.
